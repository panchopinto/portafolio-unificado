Bancos PAES de ejemplo (10 preguntas por asignatura).
- Formato JSON y CSV.
- Campos: id, pregunta, a, b, c, d, correcta, OA, PAES.

Sugerencia de integración con la app de Trivias:
- Cargar CSV con PapaParse o JSON vía fetch.
- Filtrar por asignatura/habilidad y mezclar preguntas.

Rutas ejemplo:
apps/trivias/bancos/trivias_lenguaje.csv
apps/trivias/bancos/trivias_matematica.csv
apps/trivias/bancos/trivias_ciencias.csv
apps/trivias/bancos/trivias_historia.csv
apps/trivias/bancos/trivias_ingles.csv
apps/trivias/bancos/trivias_educacion_ciudadana.csv
