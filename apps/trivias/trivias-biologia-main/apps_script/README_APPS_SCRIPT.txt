Apps Script — Instrucciones
1) Abre tu Google Sheet de alumnos → Extensiones → Apps Script.
2) Crea dos archivos y pega:
   - Code.gs (este archivo)
3) Configura SHEET_ID (ID de tu planilla) y SHEET_NAME (pestaña Alumnos).
4) Ejecuta la función setApiKey() una vez y reemplaza la clave por una fuerte.
5) Implementa como aplicación web (Ejecutar como: tú; Acceso: cualquiera con el enlace o dominio).
6) Copia la URL /exec y pégala en config.json como appsScriptUrl.
